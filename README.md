# Customer Churn Analysis

## Project Overview
This project analyzes customer churn using the IBM Telco Customer Churn dataset. The objective is to identify the factors that influence customer churn through data cleaning, exploratory data analysis (EDA), and data visualization.

## Objective
- Analyze customer churn patterns.
- Identify factors affecting customer retention.
- Provide business insights using visualizations.

## Dataset
- Dataset: IBM Telco Customer Churn
- Total Records: 7043
- Records After Cleaning: 7032
- Features: 21

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Data Cleaning
- Checked missing values.
- Converted TotalCharges to numeric format.
- Removed rows with missing values.
- Verified duplicate records.

## Visualizations
- Customer Churn Distribution
- Gender vs Churn
- Contract vs Churn
- Internet Service vs Churn
- Monthly Charges Distribution
- Customer Tenure Distribution
- Payment Method vs Churn
- Senior Citizen vs Churn
- Correlation Heatmap

## Key Insights
- Most customers stayed with the company.
- Month-to-month contracts had the highest churn.
- Fiber optic customers showed higher churn.
- Electronic Check users had higher churn.
- Long-term customers were more loyal.
- Gender had minimal impact on customer churn.

## Conclusion
Customer churn is mainly influenced by contract type, internet service, payment method, and customer tenure. Businesses can reduce churn by encouraging long-term contracts, improving service quality, and offering better customer support.
